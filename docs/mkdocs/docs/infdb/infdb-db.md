# External Tools Integration

infDB is designed to work with external tools via its standardized interfaces.

## Data Ingestion Tools

### infDB-loader
A tool specifically designed to import common open datasets (e.g., from government portals) into the infDB schemas.
-   [Documentation](../tools/infdb-loader.md)

## Analysis Tools

### pylovo
**py**thon **lo**w **vo**ltage
A tool to generate synthetic public low voltage distribution grids.

### EnTiSe
**En**ergy **Ti**me **Se**ries
A tool for generating and managing energy time series profiles.
