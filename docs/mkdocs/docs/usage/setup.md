# Setup

The setup of the infDB platform involves configuring environment variables for services and YAML files for data import.

Please select a configuration topic:

- **[Environment Variables](setup/environment-variables.md)**: Configuring the `.env` file for Docker services (ports, credentials, profiles).
- **[YAML Configuration](setup/yaml-configuration.md)**: Configuring `config-infdb-loader.yml` for data import and processing.

For detailed service-specific configuration, please refer to the [Services](../infdb/index.md#services) section.
