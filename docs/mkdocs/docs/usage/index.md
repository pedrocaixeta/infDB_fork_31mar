# Usage Guide

Welcome to the infDB usage guide for users. This section provides step-by-step instructions getting up and running with the **infDB**.

## Quick Start Overview

To get started with infDB, you will generally follow this path:

1.  [**Check Requirements**](requirements.md): Ensure your system has the necessary prerequisites.
2.  [**Get Software**](get-software.md): Clone the repository.
3.  [**Setup**](setup.md): Configure the environment variables.
4.  [**Deployment**](deployment.md): Launch the stack using Docker Compose.
5.  [**Services**](services.md): Connect to the various tools and interfaces.

Select a topic from the navigation menu to begin.
