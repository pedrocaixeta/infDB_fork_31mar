# Services
The infDB platform provides a suite of essential services designed to facilitate database operation and administration, data handling and visualization, and connectivity. Each preconfigured service can be activated individually to tailor the environment to your specific requirements.

![alt text](services.png)

## Available Services

For detailed configuration information, please refer to the dedicated service documentation linked below:

- **[infdb-db](../infdb/services/infdb-db.md)** — Central database storing all spatial, time-series, and 3D data.
- **[infdb-importer](../infdb/services/infdb-importer.md)** — Automatically downloads, validates, and integrates open datasets.
- **[pgAdmin](../infdb/services/pgadmin.md)** — Easy-to-use interface for exploring data, running queries, and managing user permissions without technical knowledge.
- **[Jupyter Notebooks](../infdb/services/jupyter.md)** — Interactive environment for exploring data, creating reports, and testing analyses with pre-configured access.
- **[QWC2](../infdb/services/qwc2.md)** — Visual map interface for viewing, analyzing, and validating spatial data directly in your browser.
- **[FastAPI](../infdb/services/fastapi.md)** — Secure access to data through simple web requests (e.g., city information, weather data).
- **[PostgREST](../infdb/services/postgrest.md)** — Automatic data access layer for reading tables and running queries without building custom code.
- **[pygeoapi](../infdb/services/pygeoapi.md)** — Standards-based interface for sharing spatial data with external GIS tools and web applications.
- **[OpenCloud](../infdb/services/opencloud.md)** — Self-hosted file share platform for managing data imports and exports.
